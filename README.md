
# Machine Learning

Machine learning is the science of making computers learn and act like humans by feeding data and information without being explicitly programmed. 



There are three subcategories of machine learning:

1.Supervised

2.Unsupervised

3.Reinforcement

## Supervised Machine Learning
In supervised learning, a model is able to predict with the help of labelled dataset.



supervised learning is basically of two types:

### 1. Regression

Regression algorithms are used when the output variable is continuous value.



Below are some popular Regression algorithm which comes under supervised learning:
* Linear Regression
* Polynomial Regression
* Decision Tree Regression
* Random Forest Regression
* Support Vector Regression

### 2. Classification

Classification algorithms are used when the output variable is categorical, which means there are two classes such as 
Yes-No, Male-Female,etc.



Below are some popular Classification algorithms which comes under supervised learning:
* Logistic Regression
* Decision Tree Classifier
* Random Forest Classifier
* Support Vector Classifier
* Naive Bayes Classifier
* K-Nearest Neighbor

## Applications of supervised learning
* Spam detection
* Stock price predictions
* Object recognition for the vision
* Fraud detection
* Image recognition
* Traffic predictions
* Self-driving car
* Medical diagnosis

## Advantages of supervised learning
* With the help of supervised learning, the model can predict the output on the basis of prior experiences.
* In supervised learning, we can have an exact idea about the classes of objects.
* Supervised learning model helps us to solve various real-world problems such as fraud detection, spam filtering, etc.

## Disadvantages of supervised learning
* Supervised learning models are not suitable for handling the complex tasks.
* It cannot predict the correct output if the test data is different from the training dataset.
* Training required lots of computation time.
* In supervised learning, we need enough knowledge about the classes of object.

## Unsupervised Machine Learning
In unsupervised learning, the algorithm is trained using data that is unlabelled.

Models itself find the hidden patterns and insights from the given data.



Unsupervised learning is basically of two types:

### 1. Clustering
Clustering is a method of grouping the objects into clusters such that objects with most similarities remains into a group and has less or no similarities with the objects of another group. 

### 2. Dimensionality Reduction
Dimensionality reduction is the process of reducing the number of random varaibles under consideration, by obtaining a set of principal varaibles.

It can be divided into feature selection and feature extraction.

#### * Feature selection
 In this, we try to find a subset of the original set of varaibles, or features, to get a smaller subset which can be used to model the problem.

#### * Feature extraction
This reduces the data in a high dimensional space to a lower dimension space, i.e. a space with lesser no. of dimensions.

#### Below is the list of some popular unsupervised learning algorithms:
* K-means clustering
* Hierarchal clustering
* Mean-shift
* DBSCAN
* principal Component Analysis

## Advantages of unsupervised learning
* It is used for more complex tasks.
* It is helpful in finding patterns in data.

## Disadvantages of unsupervised learning
* Less accuracy.
* Time consuming. 
* More the features, more the complexity.
